### Valence Helper



这是一个开源的学习软件

可以帮助你背诵《浙教版八下科学教材》中的常见元素的常见化合价

难度可调节，可以统计分数

不同的化合价用","隔开，化合价按照数值从小到大排列（不然无法识别）
